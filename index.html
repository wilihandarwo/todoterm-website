Quick Installation

Get started in seconds with your favorite package manager

📦

NPM (Recommended)

npm install -g todoterm

Works on all platforms. Requires Node.js.

🍺

Homebrew (Mac)

brew tap wilihandarwo/todoterm && brew install todoterm

Perfect for macOS users.

Quick Start

1
todoterm Launch interactive mode
2
todoterm add "My first task" Add your first todo
3
todoterm pa "My Project" Create a project

Documentation

Everything you need to master TodoTerm

📋 Basic Commands

todoterm list View todos
todoterm add "task" Add new todo
todoterm done 1 Mark as done
todoterm rm 1 Remove todo

📂 Project Management

todoterm projects List projects
todoterm pa "name" Add project
todoterm ps project-id Switch project
todoterm pr project-id Remove project

🎮 Interactive Mode

todoterm Launch interactive menu
[>] View todos
[+] Add new todo
[*] View projects
